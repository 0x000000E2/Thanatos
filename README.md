# Thanatos Ransomware

```
C:\Users\%username%\Desktop
C:\Users\%username%\Documents
C:\Users\%username%\Downloads
C:\Users\%username%\Favourites
C:\Users\%username%\Music
C:\Users\%username%\OneDrive
C:\Users\%username%\Pictures
C:\Users\%username%\Videos
```
